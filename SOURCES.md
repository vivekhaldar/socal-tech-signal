# SoCal Event Source Registry

Last updated: 2026-07-10

Use this registry before open-ended discovery. Query every proven source for the coverage window, then search broadly for new sources and events. A source being listed here does not waive canonical event-page verification.

## Maintenance

After each run:

1. Record any newly discovered recurring source only after it yields a verified event above the quality threshold.
2. Update caveats when a source produces stale dates, duplicates, virtual events disguised as local, time-zone errors, or inaccessible canonical pages.
3. Demote sources that repeatedly produce no useful candidates; do not delete their history without noting why.
4. Keep event-specific URLs out of the registry unless they are stable calendar pages. Store organizer, chapter, community, and discovery URLs instead.
